## Engineering Philosophy / Filosofi Rekayasa

- Prefer **simple, understandable systems** over opaque abstractions.
  *Mengutamakan **sistem yang sederhana dan mudah dipahami** daripada abstraksi yang rumit.*
- Stability and predictability are features, not afterthoughts.
  *Stabilitas dan prediktabilitas adalah fitur utama, bukan tambahan.*
- Systems should degrade gracefully, not fail catastrophically.
  *Sistem harus mengalami penurunan fungsi secara terukur (graceful), bukan gagal secara katastropik.*
- If a system runs unattended, it must be observable and controllable.
  *Jika sebuah sistem berjalan tanpa pengawasan, sistem tersebut harus dapat diobservasi dan dikendalikan.*
